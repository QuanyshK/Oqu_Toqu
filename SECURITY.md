## Reporting a Vulnerability

We take the security of **Oqu Toqu** seriously. If you discover a security vulnerability, please report it to us promptly.

### How to Report

Please do **not** create a public GitHub issue for security vulnerabilities. Instead, report them privately:

1.  **Email**: Send details to [INSERT CONTACT EMAIL].
2.  **GitHub Security Advisory**: If enabled, you can use the "Report a vulnerability" button in the Security tab of the repository.

### What to Include

*   A description of the vulnerability.
*   Steps to reproduce the issue.
*   Potential impact of the vulnerability.

### Our Process

1.  We will acknowledge receipt of your report within 48 hours.
2.  We will investigate the issue and determine its severity.
3.  We will work on a fix and release a security update as soon as possible.
4.  We will publicly disclose the vulnerability only after a fix has been released and users have had time to upgrade.

Thank you for helping keep Oqu Toqu secure!
